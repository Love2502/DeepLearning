## Sprint Checklist

### Sprint 1: Project Setup

- [ ] Create the virtual environment.
- [ ] Install packages from `requirements.txt`.
- [ ] Open and run the project notebook.
- [ ] Confirm all students can run the notebook kernel.

### Sprint 2: Baseline Experiments

- [ ] Train scratch CNN on species classification.
- [ ] Train scratch CNN on breed classification.
- [ ] Save first metrics and plots.
- [ ] Document design choices: layers, activations, loss, optimizer, hyperparameters.

### Sprint 3: Transfer Learning

- [ ] Train ResNet18 head on species classification.
- [ ] Train ResNet18 head on breed classification.
- [ ] Fine-tune final ResNet block if time allows.
- [ ] Compare scratch vs transfer learning.

### Sprint 4: Final Deliverables

- [ ] Run `python main.py` for the final experiments.
- [ ] Update report result tables from `outputs/metrics/model_comparison_summary.csv`.
- [ ] Add confusion matrices and training curves to report/slides.
- [ ] Export the final report and presentation PDFs.
- [ ] Check presentation length: 15 minutes plus 10 minutes Q&A.
